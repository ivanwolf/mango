# Mango

Mango is a simple react starter kit.

## What is inside

- React, React HMR.
- Webpack, Babel.
- Eslint.
- Redux, thunks, logger and devtools.
- Minimal [Bulma](https://bulma.io) setup.
- [glamorous](https://glamorous.rocks/) brings inline styles.
